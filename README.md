UpdateResolv
=========

Simple role to update namserver on server with /etc/resolv.conf

Requirements
------------

None

Role Variables
--------------

You must define which nameserver you want to use on defaults/main.yml change values of namservers, it is seted to
google dns's by default.

Dependencies
------------

None
